# Text Information Retrieval System

**Indexed Search Engine for Controlled Digital Library**

## Project Overview

A full-stack information retrieval system that indexes a controlled collection of books and retrieves the most relevant pages for a given text query using inverted index and TF-IDF ranking.

## Problem Statement

Given an input text query (sentence or paragraph), the system identifies the source book and page number from a controlled digital library.

## Team Members

- Member 1: Backend + Algorithms
- Member 2: Frontend + Auth + Documentation

## Tech Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **OCR**: Tesseract.js (for scanned PDF text extraction)
- **Algorithm**: Inverted Index + TF-IDF Ranking

## Project Structure

```
/backend        - Express server, API, algorithms
/frontend       - React application
/docs           - All documentation
/scripts        - Utility scripts
/dataset        - Dataset metadata (PDFs not committed)
```
